Logging System
--------------

- Uses `Loguru`_ for structured, colored logging
- Each node has its own log buffer
- ``self.logger.info("...")`` captures output
- ``print()`` output is automatically captured
- Logs saved per-node and visible in UI

Data Flow Rules
---------------

- Inputs auto-populate from upstream outputs
- Outputs stored by port name (dict key)
- Supports multiple inputs/outputs
- No circular dependency allowed
