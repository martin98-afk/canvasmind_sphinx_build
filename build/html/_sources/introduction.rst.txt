============
Introduction
============

A modern low-code visual programming platform built on **NodeGraphQt** and **qfluentwidgets**, supporting:

- Drag-and-drop component orchestration
- Asynchronous execution
- File operations, loop & branch control
- One-click export of workflows into standalone runnable projects

Seamlessly transition from prototype to production.
